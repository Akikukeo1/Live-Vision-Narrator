# src-go

このディレクトリには、Goで実装されたコードが配置されます。GoはAPIの主要なコードベースとして使用されます。Goコードは、Pythonから移行された機能を含むすべての新しい機能を実装するために使用されます。Goコードは、Pythonコードと同様に、APIのロジック、データ処理、および外部サービスとの統合を担当します。Goコードは、Pythonコードが廃止されるにつれて、完全に置き換えることを目指しています。

すべての新機能はGoのみに実装されます。Pythonコードは現在参照用として残されていますが、新しい機能はGoコードに追加され、PythonAPIが正しく機能するかは保証されません。
